### List - ArrayList

#### ArrayList 的特征

Manipulation with ArrayList is slow because it internally uses and array. 在删减元素时，其他元素会需要进行 shift 操作

Better for storing and accessing data

存储 ArrayList 的内存位置是连续的

<br></br>
#### Limitations of ArrayList

The underlying array has fixed capacity

time consuming for capacity incresement

Cannot easily insert a value between two others

Searching for a value can be very slow （查找是否存在某个元素）

### List - LinkedList

#### LinkedList 的特征

- Manipulation with LinkedList is faster than ArrayList. No bit shifting is required.
- LinkedList can act as a list and queue both because it implements List and Deque inetrfaces
- LinkedList is better for manipulating
- The first node of the linked list is called the Head.
- The next pointer of the last node always points to NULL.
- In a linked list, insertion and deletion are possible easily.
- Linked lists can shrink or grow at any point in time easily.

<br></br>
#### LinkedList 的特征 *ArrayList* : store data in adjacent memory space - continuously stored

- LinkedList\* : store data in memory space arbitarily but linked in some way.
  - Additional piece of information indicating the sequence (used to link elements together)
